== Description ==
Facebook like box for wordpress is plugin that allow you to customize facebook like box easily. This is very useful plugin for websites, communities that use facebook and want to share there facebook page on their website to get more likes on facebook. This is nice plugin, all in one. You can customize every thing you want. There are a lot of features you can do with this plugin. You can change all facebook original features like: title, width, height, color scheme and more. But in this plugin there is add some more features like: cookies, overlay and styles.

== Installation ==
Upload the entire facebook-like-box/ folder to the /wp-content/plugins/ directory.
Activate the plugin through the \'Plugins\' menu in WordPress.
Customize the plugin through the \'FB Like Box\' menu in WordPress.
